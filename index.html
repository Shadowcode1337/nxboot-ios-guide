What is NXBoot?

NXBoot is an app developed for IOS and macOS to start custom boot code trough the Fusee Gelee hardware exploit.

Requirements

  • A jailbroken iPhone

  • IOS 10.0+

  • Cydia

Hardware requirements

  • A Lightning OTG adapter

    Instructions

    Section 1

    1. Open Cydia and go to the source tab

    2. Click on edit on the top right then click add and enter this url https://mologie.github.io/repo/ it should look like this.

    3.
    Click on "Mologie".

    4. "All Packages".

    5. Click on NXBoot and Install it on the top right corner

    6. After installation you should see the nxboot app on your homescreen
